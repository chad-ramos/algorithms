# Dynamic Array

| Operation | Complexity |
|---|---|
| Access  | O(1) |
| Search | O(n) |
| Insertion |  O(n) |
| Appending | O(1) |
| Deletion | O(n) |

# Linked List
| Operation | Complexity |
|---|---|
| Search  | O(n) |
| Insert At Head | O(1) |
| Insert At Tail |  O(1) |
| Remove At Head | O(1) |
| Remove At Tail | O(n) |
| Remove At Middle | O(n) |
